# Stelltask: Stellar-Native Open Work Platform

## Core MVP Structure

- **apps/web**: Next.js 14 (App Router) frontend.
- **apps/api**: NestJS 10 backend.
- **packages/types**: Shared TypeScript definitions.
- **packages/stellar-utils**: Shared Stellar SDK helpers.

## Getting Started

1. `pnpm install` (once dependencies are added)
2. `docker-compose up -d`
3. `pnpm dev`
